# 福瑞勇者从不战败 - Web 原型

这是《福瑞勇者从不战败》的 Web 原型仓库。

第一版原型用于验证核心循环：

1. 播放一段剧情。
2. 进入卡牌交互。
3. 通过卡牌交互触发下一段剧情。

## 当前范围

第一版可玩切片计划覆盖：

- `1-1`：剧情开场、主角卡、技能卡 `我`、强制拖拽教学、取名框。
- `1-2`：生成 `树`、`草地`、`纹身`、`双手` 等探索卡牌，通过卡牌交互触发发现黑日的剧情，并推进到哈维登场。

原型采用数据驱动设计。后续确定要做编辑器，因此剧情段、卡牌定义、交互规则都应保持结构化，不写死在 UI 组件里。

## 重要产品备注

- 卡牌进度条是可选设计，只用于部分有读条、蓄力、进度反馈需求的卡牌。
- 普通卡牌不显示进度条。
- 图像生成只负责生成卡牌“牌面插画”。
- 卡牌名、卡牌类型、持有数量、进度条、卡牌描述等可变内容由前端 UI 渲染。
- 编辑器是后续确定要做的功能，但不包含在第一版原型中。

```bash
npm run dev
```

默认本地地址为 <http://127.0.0.1:5173/>。

